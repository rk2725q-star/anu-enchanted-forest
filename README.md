### ✨ Key Features
- **🌿 Enchanted UI:** Immersive forest-themed experience with wood panels and glowing accents.
- **🧠 Persistent Memory:** Anu remembers your health, family, and past conversations forever.
- **🎙️ Multimodal:** Send voice notes, files, or images—Anu understands them all.
- **🎨 Magical Art:** Generate high-quality AI art directly within the conversation.
- **📞 Direct Link:** Optimized for both local GPU power and cloud-based Vercel deployment.

---

### 🚀 Quick Start (Local)

**Prerequisites:** Node.js (v18+)

1. **Install Dependencies:**
   ```bash
   npm install
   ```
2. **Setup Env:** Create a `.env` file with:
   ```env
   GEMINI_API_KEY=your_key_here
   ```
3. **Run Dev Server:**
   ```bash
   npm run dev
   ```

### ☁️ Deployment
This project is pre-configured for **Vercel**. 
1. Push to your GitHub repo.
2. Link the project in Vercel Dashboard.
3. Set `GEMINI_API_KEY` in Environment Variables.
